# ENG22CS0354-Enhanced-Seizure-Prediction-Using-SMOTE-and-Ensemble-Machine-Learning-Models-on-EEG-Data
This project improves seizure prediction from EEG data using SMOTE to handle class imbalance. Ensemble models like Random Forest and XGBoost enhance accuracy and reliability. It enables early detection of seizures, supporting real-time monitoring and better patient care.
🧠 Enhanced Seizure Prediction Using SMOTE & Ensemble ML
📌 Overview

This project focuses on predicting epileptic seizures using EEG data and advanced machine learning techniques. It addresses class imbalance using SMOTE and improves prediction accuracy using ensemble models like Random Forest and XGBoost.

🚀 Features

EEG data preprocessing and feature scaling

Handling imbalanced data using SMOTE

Ensemble learning (Random Forest, XGBoost)

Model evaluation using accuracy, precision, recall, F1-score

Threshold tuning for better seizure detection
🛠️ Tech Stack

Python

NumPy, Pandas

Scikit-learn

XGBoost

Imbalanced-learn (SMOTE)

Matplotlib / Seaborn (for visualization)
