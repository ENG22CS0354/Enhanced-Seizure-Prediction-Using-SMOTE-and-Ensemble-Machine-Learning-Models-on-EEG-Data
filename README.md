# ENG22CS0354-Enhanced-Seizure-Prediction-Using-SMOTE-and-Ensemble-Machine-Learning-Models-on-EEG-Data
This project improves seizure prediction from EEG data using SMOTE to handle class imbalance. Ensemble models like Random Forest and XGBoost enhance accuracy and reliability. It enables early detection of seizures, supporting real-time monitoring and better patient care.
